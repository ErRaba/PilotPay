# PilotPay

Calculadora y auditor de nóminas para pilotos de Binter Canarias.

## Arquitectura

Proyecto dividido en:

- frontend/ → interfaz de usuario
- backend/ → motor de cálculo auditable

## Instalación

### Backend

cd backend  
npm install  
npm start  

### Frontend

Abrir:

frontend/index.html

## Estado

- Pilotos: funcional y en validación avanzada
- TCP: pendiente de validación con nóminas reales

## Objetivo

Auditoría precisa de nóminas conforme a convenio y legislación española.
